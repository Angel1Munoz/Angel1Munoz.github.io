What is it?
Origami is an endless game that you survive for as long as you can seeing how high your score can get. Dodge enemy fire and try to hit them to destroy them. As the game progresses, it gets faster and faster making the game a lot more difficult. In the video we can see the user playing the game on the selected difficutly which is medium and coninues to get points but then shows the mechanics of losing and how the score and highscore are displayed in the menu. The input of the program is whatever the user decides to chose at the beginning difficult wise and then how the user decides to move their character and fire. The output being how the game is running and the reactions of the enemies.
